# Supabase SQL (Deprecated)

The SQL files in this directory were originally intended for a Supabase-backed implementation.
This project no longer uses Supabase; the files are kept for historical reference only and are not
required for running the current Convex-based application.

If you are migrating from a legacy Supabase deployment, review these files and extract any
required SQL manually. No automated Supabase setup is provided.
